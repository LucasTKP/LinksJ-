### 💻 Projeto:

Este projeto consiste em um sistema para agrupar os links de todas as redes socias, bem parecido com um linktree.

<br /> <br />

### 🎯 Objetivo:

Criar um site que exibe todas as redes sociais de uma empresa.


<br /> <br />
